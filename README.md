# ГЭЙм — Factory Automation Game

2D factory automation game inspired by Factorio and Mindustry. Built with HTML, CSS, and JavaScript Canvas.

## How to Play

**Movement:** WASD / Arrow keys  
**Collect ore:** Walk over ore tiles or press E  
**Rotate direction:** Q  
**Build conveyor:** F / Left-click (cost: 1 iron ore)  
**Build furnace:** G / Right-click (cost: 3 iron ore)  
**Build drill:** H (cost: 5 iron ore)  
**Build assembler:** J (cost: 8 iron ore)  
**Save:** K  
**Load:** L  
**Reset:** R  

## Gameplay

1. Collect iron and copper ore by walking over it
2. Build conveyor belts to transport items
3. Build furnaces to smelt ore into plates (5 ore → 1 plate)
4. Build drills to automate ore mining (feeds onto belts)
5. Build assemblers to craft circuit boards (2 iron plate + 2 copper plate → 1 circuit board)
6. Chain belts → furnaces → belts → assemblers for full automation
